About This App

I built this for my INST 377 final project. The idea came from how annoying it is to have like 5 different tabs open when you are planning a trip. This app puts weather, currency conversion, and flight status all in one place.

APIs Used

Weatherstack - gives current weather data for any city

Open Exchange Rates - gives live currency exchange rates, no key needed

Aviationstack - gives real time flight status by flight number

Libraries Used

Chart.js - used on the weather page to show a bar chart of weather data

Anime.js - used on the weather page to animate the results when they load
